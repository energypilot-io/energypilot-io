# Contributing to EnergyPilot.io

Thanks for taking the time to read this!

The `EnergyPilot.io` project is in early development, therefore we encourage contribution in the following ways:

## Documentation

That's one thing to do.

## Bug Reports

Please report flaws or vulnerabilities in the [GitHub Issue Tracker](https://github.com/energypilot-io/energypilot-io/issues) using the corresponding issue template.

## Ideas & Features

Issues in the [GitHub Issue Tracker](https://github.com/energypilot-io/energypilot-io/issues) are also fine
to discuss ideas and features.

## Code Contributions

We welcome code contributions and bug fixes via [Pull Requests](https://github.com/energypilot-io/energypilot-io/pulls).
To make collaboration easier, we require pull requests to pass code style, unit tests, and commit
message style checks.

## Thank you!

And last but not least thanks to all our contributors
